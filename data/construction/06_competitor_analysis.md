## クラスター1: 採用・人材確保（建設業）

### 対象課題
- 施工管理技士の応募が来ない
- 職人の高齢化・後継者不足
- 若手の早期離職
- 清掃員・設備員の採用困難

### 競合マップ

#### 1-1. 大手求人媒体

**Indeed**
- URL: indeed.com
- 提供内容: 求人掲載（無料+有料スポンサー）
- 対象顧客: 全業種・全規模
- 価格帯: クリック課金型（100円〜1,000円/クリック）
- 弱点（根拠付き）:
  - 建設業特化の知見がない（根拠: 汎用プラットフォームのため業界別の最適化機能なし）
  - 求人票の差別化が困難（根拠: フォーマットが画一的、施工実績等の訴求がしにくい）
  - クリック単価が高騰傾向（根拠: 建設業の有効求人倍率5倍超で競争激化）

**doda**
- URL: doda.jp
- 提供内容: 求人掲載 + 人材紹介
- 対象顧客: 中堅〜大手企業中心
- 価格帯: 掲載4週間25万〜150万、紹介は年収35%
- 弱点（根拠付き）:
  - 中小建設会社には高額（根拠: 最低掲載料25万/4週間、専門工事の社長には負担大）
  - ホワイトカラー求職者が多く、現場技術者のマッチ率が低い（根拠: 登録者の職種分布）

#### 1-2. 建設業特化求人

**建設転職ナビ（ヒューマンタッチ）**
- URL: kensetsutenshokunavi.jp
- 提供内容: 建設技術者専門の転職支援
- 対象顧客: 中堅〜大手建設会社
- 価格帯: 人材紹介（年収30〜35%）
- 弱点（根拠付き）:
  - 中途採用の即戦力がメイン、未経験採用には不向き（根拠: 掲載求人の大半が有資格者向け）
  - 紹介料が高額（根拠: 年収500万×35%=175万/人）
  - 地方の中小企業への紹介実績が薄い（根拠: 求人が首都圏・大阪圏に偏重）

**助太刀**
- URL: suke-dachi.jp
- 提供内容: 職人マッチングアプリ
- 対象顧客: 専門工事業者、職人
- 価格帯: 月額課金制
- 弱点（根拠付き）:
  - 正社員採用ではなくスポットマッチングが主（根拠: アプリの機能設計が日雇い・短期前提）
  - 登録職人の質にばらつき（根拠: アプリレビューでの指摘多数）
  - 継続的な人材確保の仕組みではない（根拠: 案件単位のマッチング設計）

#### 1-3. 人材派遣

**テクノプロ・コンストラクション / 夢真**
- 提供内容: 施工管理技術者の派遣
- 対象顧客: ゼネコン、サブコン
- 価格帯: 月額50万〜80万/人
- 弱点（根拠付き）:
  - コストが高い（根拠: 自社採用の1.5〜2倍のコスト）
  - 派遣技術者の定着・帰属意識の問題（根拠: 現場間の異動が頻繁）
  - 自社の採用力は向上しない（根拠: 派遣依存が続く構造）

#### 1-4. 採用マーケティング・ブランディング

**engage（エン・ジャパン）**
- 提供内容: 無料採用HP作成ツール
- 対象顧客: 中小企業全般
- 価格帯: 基本無料、有料プランあり
- 弱点（根拠付き）:
  - テンプレートが汎用的で建設業の魅力が伝わりにくい（根拠: 現場写真・施工実績の訴求テンプレなし）
  - SEOでの集客力が限定的（根拠: engage内での検索が主、Google検索からの流入は弱い）

### クラスター1 競合全体の弱点まとめ

| 弱点カテゴリ | 内容 | 根拠 |
|-------------|------|------|
| 対象企業規模の偏り | 大手・中堅向けが多く、従業員30人以下の専門工事・工務店に弱い | 掲載料・紹介料の価格帯が中小に合わない |
| 導入の重さ | 人材紹介は1人採用に100万〜200万かかる | 紹介手数料の相場 |
| 建設業の理解不足 | 汎用媒体は「施工管理」と「現場監督」の違いすら曖昧 | 求人カテゴリの粒度が粗い |
| 自社採用力の構築支援がない | 媒体依存から脱却できない | 媒体のビジネスモデル上、自社採用力向上は利益相反 |
| 採用後の定着支援がない | 採用して終わり | 入社後のオンボーディング・定着支援サービスが皆無 |

## クラスター2: 集客・問い合わせ獲得（工務店・リフォーム）

### 対象課題
- 新規問い合わせが少ない
- 見学会の集客減少
- 大手ハウスメーカーとの差別化困難
- SNS・ブログの更新が続かない
- Web問い合わせからの成約率が低い

### 競合マップ

#### 2-1. 住宅ポータルサイト

**SUUMO（リクルート）**
- URL: suumo.jp
- 提供内容: 住宅メーカー・工務店の掲載、資料請求誘導
- 対象顧客: ハウスメーカー、工務店、リフォーム会社
- 価格帯: 月額5万〜50万（掲載プランによる）
- 弱点（根拠付き）:
  - 資料請求の質が低い（根拠: 一括資料請求により「とりあえず請求」が多い）
  - 大手ハウスメーカーが上位表示される（根拠: 広告費用の多い企業が優遇される掲載ロジック）
  - 掲載をやめると問い合わせがゼロに（根拠: 自社への資産蓄積がない構造）
  - 地域密着工務店の差別化が難しい（根拠: 画一的な掲載フォーマット）

**HOME'S / LIFULL HOME'S**
- URL: homes.co.jp
- 提供内容: 住宅会社掲載、資料請求
- 対象顧客: 同上
- 価格帯: 同等レベル
- 弱点: SUUMOと同様の構造的課題

#### 2-2. 工務店向けコンサル・FC

**ジョンソンパートナーズ**
- 提供内容: 工務店の集客・経営コンサルティング、FC展開
- 対象顧客: 地域工務店（年間10〜50棟クラス）
- 価格帯: 月額10万〜50万 + 成果報酬型の場合もあり
- 弱点（根拠付き）:
  - コンサル費用が継続的にかかる（根拠: 月額課金モデル）
  - FC加盟の場合、自社ブランドが薄まる（根拠: FC本部のブランド使用が前提）
  - 成果にばらつきがある（根拠: 地域性・既存ブランド力の差で効果が異なる）

**新建ハウジング / 住宅産業研究所**
- 提供内容: 業界メディア、マーケティング支援
- 対象顧客: 工務店経営者
- 価格帯: セミナー参加費、情報誌購読
- 弱点（根拠付き）:
  - 情報提供が主であり実行支援が薄い（根拠: メディア・情報事業がメイン）
  - 実行は結局自社でやる必要がある（根拠: 代行サービスではない）

#### 2-3. Web制作・マーケティング

**一般Web制作会社**
- 提供内容: HP制作、SEO対策
- 対象顧客: 全業種
- 価格帯: 初期50万〜300万 + 月額保守1万〜5万
- 弱点（根拠付き）:
  - 建設業・住宅業界の知見がない（根拠: 汎用の制作会社が大多数）
  - 作って終わりになりがち（根拠: 保守契約は更新・セキュリティのみで、集客改善は別料金）
  - 成果（問い合わせ数）にコミットしない（根拠: 制作費モデル）

**住宅業界特化Web制作**
- 提供内容: 工務店向けHP・集客支援
- 対象顧客: 工務店、リフォーム会社
- 価格帯: 初期100万〜500万 + 月額5万〜20万
- 弱点（根拠付き）:
  - 高額（根拠: 住宅特化の分、一般Web制作より割高）
  - SNS・コンテンツ運用は別契約が多い（根拠: Web制作と運用が分離）
  - 小規模工務店には投資回収が困難（根拠: 年間10棟以下だとROIが合わない）

#### 2-4. SNS運用代行

**一般SNS運用代行会社**
- 提供内容: Instagram、YouTube等の運用代行
- 対象顧客: 全業種
- 価格帯: 月額10万〜50万
- 弱点（根拠付き）:
  - 住宅・建設の施工写真撮影ノウハウがない（根拠: 現場の撮影は自社でやる必要がある）
  - 投稿内容が薄い（根拠: 業界知識がないと「映える」だけで訴求力がない）
  - 問い合わせまでの導線設計が弱い（根拠: フォロワー増加が目的化し成約との接続が不明確）

### クラスター2 競合全体の弱点まとめ

| 弱点カテゴリ | 内容 | 根拠 |
|-------------|------|------|
| 媒体依存構造 | 掲載をやめると成果がゼロ | ポータルサイトの構造上、自社資産にならない |
| 導入コストが高い | Web制作+運用で年間200万〜 | 小規模工務店にはROIが合わない |
| 実行支援の不足 | コンサルは「やり方」は教えるが「やる」のは自社 | 忙しい工務店が実行しきれない |
| 統合的な仕組みがない | HP、SNS、広告、イベントがバラバラ | 複数の業者を使い分ける手間 |
| 成果指標が不明確 | PV、フォロワー等の中間指標で終わる | 問い合わせ→商談→受注までの一気通貫が見えない |
